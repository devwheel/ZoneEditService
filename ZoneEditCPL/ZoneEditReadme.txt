The zoneedit updater program uses code that I have written and borrowed 
from others to give one consolidated package that runs the updater as a windows
service with configuration of the service added to the control panel.

File List
---------------------------------------------------------
ZoneEdit.dll......... Library for updating configuration
ZoneEdit.cpl......... Control panel applet - launches ZoneEditCPL.exe
ZoneEditCPL.exe...... Configuration Tool
Setup.Exe............ Setup Program
ZoneEditSvc.config... Configuration File
ZoneEditSvc.exe...... Windows Service File
InstallUtil.exe...... For Installing the Service
InstallUtilLib.dll... Used by InstallUtil.exe
ZoneEditReadme.txt... This file
